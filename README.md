# Demand Forecasting Platform

AI-powered demand forecasting web application built using Django, Machine Learning, Bootstrap, and Python.

---

## Features

- CSV File Upload
- AI Demand Prediction
- Product Demand Analysis
- Forecast Dashboard
- Interactive Charts
- Stock Refill Recommendations
- Business Insights
- Analytics Reports

## Installation

Clone repository:

```bash
git clone https://github.com/bhuvneshwari-07/demand-forecasting-platform.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run server:

```bash
python manage.py runserver
```
